# MNIST Handwritten Digit Classification

This project trains a feedforward neural network to classify handwritten digits (0-9) from the MNIST dataset using PyTorch. The code is structured for clarity and includes visualizations to showcase results. Explanations for each code block are provided in English and Russian.

**Русский**: Этот проект обучает полносвязную нейронную сеть для классификации рукописных цифр (0-9) из датасета MNIST с использованием PyTorch. 
Код структурирован для ясности и включает визуализации для демонстрации результатов. Объяснения для каждого блока кода предоставлены на английском и русском языках.

## Features
- Neural network with two hidden layers and dropout for regularization.
- Visualizations: Accuracy plot, sample predictions, and confusion matrix.
- Test accuracy: ~95-97%.


## How to Run
1. Download this project from GitHub.
2. Install libraries: `pip install -r requirements.txt`
3. Open `MNIST_Classification.ipynb` in VS Code and run it.
